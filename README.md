# 🍽️ FlavorVerse — Recipe Sharing Website

A full-stack recipe sharing platform where users can discover, create, and manage recipes with meal planning and shopping list features.

## ✨ Features

- 🔐 User Authentication — Register and login with JWT security
- 🍳 Recipe Management — Create, edit, delete recipes with image upload
- ⭐ Recipe Rating System — Rate recipes 1 to 5 stars
- 🔍 Ingredient-Based Search — Search recipes by ingredients
- 📅 Weekly Meal Planner — Plan your meals for the week
- 🛒 Smart Shopping List — Auto-generate shopping list from meal planner
- 🥗 Nutritional Information — View calories, protein, carbs, fat per serving
- ❤️ Favorites — Save your favourite recipes

## 🛠️ Tech Stack

Frontend: React 19, React Router v7, Axios, Tailwind CSS, Vite
Backend: Node.js, Express.js, MongoDB, Mongoose, JWT, Multer

## 🚀 Getting Started

### 1. Clone the repository
git clone https://github.com/harshitarao7771-droid/recipe-sharing-website.git

### 2. Start Backend
cd server
npm install
npm start

### 3. Start Frontend
cd ..
npm install
npm run dev

### 4. Open in browser
http://localhost:5173
